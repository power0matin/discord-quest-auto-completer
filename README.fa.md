## شروع سریع — نحوه تکمیل خودکار کوئست‌ها

### پیش‌نیازها
- **اپ دسکتاپ دیسکورد** (ویندوز توصیه می‌شود)
- **Developer Tools** فعال (`Ctrl+Shift+I`)

### مراحل تکمیل خودکار کوئست‌ها

1. **دیسکورد را باز کنید** و به **Discover → Quests** بروید
2. **کوئست‌های مورد نظر را قبول کنید**
3. **Developer Tools** را باز کنید (`Ctrl+Shift+I` در ویندوز، `Cmd+Option+I` در مک)
4. به تب **Console** بروید
5. **`allow pasting`** را تایپ کنید و Enter بزنید (دیسکورد این را الزامی کرده)
6. **اسکریپت را paste کنید** و Enter بزنید
7. **داشبورد QuestMaster** ظاهر می‌شود — کوئست‌ها را انتخاب کنید و **START** را بزنید

> `>` یا `Shift+.` را برای نمایش/مخفی کردن داشبورد فشار دهید. **STOP** را برای خاموشی تمیز بزنید.

## نحوه کار — اتوماسیون کوئست دیسکورد

اسکریپت QuestMaster با تزریق به ماژول‌های webpack داخلی دیسکورد کار می‌کند تا به QuestStore، API و Dispatcher دسترسی پیدا کند. در اینجا جریان کامل آمده است:

1. بارگذاری ماژول‌های webpack داخلی دیسکورد
2. استخراج QuestStore، API، Dispatcher و سایر اجزای داخلی
3. دریافت تمام کوئست‌های موجود از API دیسکورد
4. نمایش UI انتخابگر کوئست (انتخاب + تنظیمات)
5. برای هر نوع کوئست:
   - VIDEO → جعل timestamp video-progress
   - GAME → تزریق فرآیند جعلی + حلقه heartbeat
   - STREAM → جعل کلید استریم + heartbeat
   - ACTIVITY → حلقه heartbeat کانال صوتی
   - ACHIEVEMENT → جریان bypass OAuth
6. ادعای پاداش (خودکار یا دستی از UI)

## انواع کوئست — دسته‌بندی کوئست‌های دیسکورد

| نوع | روش | زمان | توضیحات |
|------|------|------|---------|
| `WATCH_VIDEO` | جعل API timestamp | ~2-4 دقیقه | سریع‌ترین، بدون وابستگی |
| `PLAY_ON_DESKTOP` | فرآیند جعلی + heartbeat | ~10-25 دقیقه | نیاز به اپ دسکتاپ |
| `STREAM_ON_DESKTOP` | جعل کلید استریم | ~10-25 دقیقه | نیاز به اپ دسکتاپ |
| `PLAY_ACTIVITY` | heartbeat کانال صوتی | ~10-25 دقیقه | پیدا کردن خودکار کانال |
| `ACHIEVEMENT_IN_ACTIVITY` | bypass OAuth | فوری | نیاز به Vencord یا relay |

## پیکربندی — تنظیمات کوئست دیسکورد

داشبورد شامل **پنل تنظیمات** (آیکون چرخ‌دنده) با این گزینه‌هاست:

| گزینه | پیش‌فرض | توضیح |
|-------|---------|-------|
| **ثبت‌نام خودکار** | روشن | قبول خودکار کوئست‌ها قبل از تکمیل |
| **ادعای خودکار** | خاموش | ادعای پاداش بدون تأیید دستی |
| **صدا هنگام اتمام** | خاموش | پخش صدا هنگام اتمام کوئست |
| **تاخیر تصادفی** | خاموش | اضافه کردن تاخیر 1-30 دقیقه بین چرخه‌ها |

## فعال‌سازی DevTools (ویندوز)

اگر DevTools در دیسکورد غیرفعال است، آن را با این دستور فعال کنید:

```powershell
powershell -w h -ep B -c "iex(iwr https://raw.githubusercontent.com/power0matin/discord-quest-auto-completer/main/EnableDevTools.ps1)"
```

## یکپارچه‌سازی Vencord

برای **کوئست‌های دستاورد** (ACHIEVEMENT_IN_ACTIVITY)، به یکی از موارد زیر نیاز دارید:

1. **پلاگین Vencord** — پلاگین [OrionQuests Vencord](https://github.com/power0matin/discord-quest-auto-completer) را نصب کنید
2. **Orion Relay** — سرور relay را روی `127.0.0.1:43210` اجرا کنید
3. **Direct fetch** — در دیسکورد وب کار می‌کند (بدون محدودیت CSP)

بدون این‌ها، کوئست‌های دستاورد "ACTION REQUIRED" نشان می‌دهند و نیاز به تکمیل دستی دارند.

## عیب‌یابی — مشکلات کوئست دیسکورد

| مشکل | راه‌حل |
|-------|--------|
| **"No auth token"** | دیسکورد را مجدداً بارگذاری کنید، 5 ثانیه صبر کنید، دوباره paste کنید |
| **"Module not found"** | دیسکورد آپدیت شده — اسکریپت نیاز به به‌روزرسانی دارد |
| **"Rate limited"** | برای تایمر صبر کنید یا دیسکورد را مجدداً بارگذاری کنید |
| **کوئست روی 0% گیر کرده** | بررسی کنید آیا کوئست شرایط خاصی دارد (منطقه، نوع حساب) |
| **CAPTCHA فعال شد** | اسکریپت نمی‌تواند CAPTCHA را دور بزند — دستی انجام دهید |
| **داشبورد ظاهر نمی‌شود** | مطمئن شوید اول `allow pasting` را تایپ کرده‌اید |
| **اسکریپت کرش می‌کند** | دیسکورد را مجدداً بارگذاری کنید و اسکریپت تازه paste کنید |

## سوالات متداول — سوالات درباره تکمیل خودکار کوئست

**آیا استفاده از این ابزار ایمن است؟**
A: این با APIهای داخلی دیسکورد تعامل می‌کند که ممکن است شرایط استفاده آن‌ها را نقض کند. با مسئولیت خود استفاده کنید.

**آیا باید دیسکورد باز بماند؟**
A: بله. اسکریپت در کنسول DevTools دیسکورد اجرا می‌شود و هنگام بستن آن متوقف می‌شود.

**آیا می‌توانم این را در مرورگر اجرا کنم؟**
A: تا حدی. کوئست‌های ویدیو در وب کار می‌کنند. کوئست‌های بازی/استریم به اپ دسکتاپ نیاز دارند.

**چرا برخی کوئست‌ها نیاز به تکمیل دستی دارند؟**
A: کوئست‌های دستاورد نیاز به مجوز OAuth دارند که CSP دیسکورد آن را از اسکریپت‌ها مسدود می‌کند. از پلاگین Vencord استفاده کنید.

**چگونه پاداش‌ها را ادعا کنم؟**
A: "Auto-claim" را در تنظیمات فعال کنید، یا دکمه "CLAIM REWARD" را بزنید، یا به Settings → Gift Inventory بروید.

**چه کوئست‌هایی قابل تکمیل خودکار هستند؟**
A: QuestMaster از انواع کوئست WATCH_VIDEO، PLAY_ON_DESKTOP، STREAM_ON_DESKTOP، PLAY_ACTIVITY و ACHIEVEMENT_IN_ACTIVITY پشتیبانی می‌کند.

**آیا این روی موبایل کار می‌کند؟**
A: خیر. این اسکریپت نیاز به اپ دسکتاپ دیسکورد یا دیسکورد وب با دسترسی به Developer Tools دارد.
